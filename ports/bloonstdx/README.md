## Controls

| Button | Action |
|--|--| 
|DPAD / Left Stick|Mouse movement|
|A|Mouse Click|
|B/Select|Deselect|
|X|Slow mouse|
|Y|Start next round/speed up|
|R1/L1/L2|Monkey upgrades|
|Start|Enter the game|
