\## How to make it work



First go to https://nicmagnier.itch.io/unlevel and install the .love file then place that .love file in ports\\unlevel\\gamedata then it should work!





\## Controls



| Button | Action |

| ------ | ------ |

| D-PAD  | Move   |
| A/B  | Tilt level |

| A      | Space  |

| R1     | Level Change  |

| START  | Proceed/tilt  |
