**Want to try porting? Looking for an easy place to start?**

Here are 7 Kultisti (https://kultisti.itch.io) GameMaker games. I had to apply a few patches to get some of them running, but they all work now. We can distribute any of these as ready-to-run. I avoided games that are highly mouse-reliant or didn't seem an obvious good fit for handhelds.

One of the main challenges for new PortMaster contributors isn’t the port itself -- it’s getting the packaging right, becoming familiar with our processes, and submitting a successful GitHub pull request that adheres to the project guidelines.

If you’d like to take one on, pick a game, decide whether it’s worth including in our catalogue, test it, refine it, start a testing-n-dev thread, and prepare a proper package.

For reference, here's a compliant Kultisti port that resides in our GitHub repo:
```
https://github.com/PortsMaster/PortMaster-New/tree/8cca10c1e21837ed1ce55d264cd8a2854f0950b7/ports/hopowl

(but you'll want to use the v4 JSON generator here: https://portmaster.games/port-json.html)
```
