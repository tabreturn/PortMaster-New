## Controls

| Button     | Action                |
| ---------- | --------------------- |
| *Menu*     |                       |
| UP/DOWN    | Up/down one entry     |
| LEFT/RIGHT | Page up/down          |
| SELECT     | Quit                  |
| *In game*  |                       |
| D-PAD      | Move                  |
| X          | Action                |
| B          | Undo                  |
| R1         | Restart               |
| SELECT     | Quit to selector menu |
