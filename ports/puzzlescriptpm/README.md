## Controls

### Selector menu

| Button     | Action             |
| ---------- | ------------------ |
| UP/DOWN    | Up/down one entry  |
| LEFT/RIGHT | Page up/down       |
| SELECT     | Quit               |

### In game

| Button | Action                 |
| ------ | ---------------------- |
| D-PAD  | Move                   |
| X      | Action                 |
| B      | Undo                   |
| R1     | Restart                |
| SELECT | Quit to selector menu  |
