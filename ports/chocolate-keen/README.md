## Notes

Thanks to [jamesfmackenzie](https://github.com/jamesfmackenzie/chocolatekeen) for this open-source reverse engineered source port of Commander Keen.  
Modded for handhelds: [Slayer366](https://github.com/slayer366/chocolatekeen).  

Get Commander Keen from:  
[Steam](https://store.steampowered.com/app/9180/Commander_Keen/)  
OR  
[GOG](https://www.gog.com/en/game/commander_keen_complete_pack)  

#### Game files needed:

**Keen 1 (Marooned on Mars)**:
Commander Keen 1: Marooned on Mars is shareware and is included and ready to play.

**Keen 2 (The Earth Explodes)**:
Place all files for 'The Earth Explodes' (CK2, etc.) in 
chocolate-keen/GAMEDATA/KEEN2

**Keen 3 (Keen Must Die!)**:
Place all files for 'Keen Must Die!' (CK3, etc.) in 
chocolate-keen/GAMEDATA/KEEN3

</br>

## Controls

| Button | Action |
|--|--| 
|Select/L1|Esc|
|Start/R1|Enter|
|A|Jump|
|B|Fire|
|X|Pogo Stick|
|Y|Answer 'Yes'|
|L2|Quick Save|
|R2|Space/Stats/Inventory|
|Up|Move Up|
|Down|Move Down|
|Left|Move Left|
|Right|Move Right|
|Left Analog/Right Analog|Move/Same as D-Pad|

</br>

## Compile
```shell
git clone https://github.com/slayer366/chocolatekeen
cd chocolatekeen/build/linux
chmod +x build_linux.sh
./build_linux.sh
```
