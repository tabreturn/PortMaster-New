   	====================
  	General Instructions
 	====================

This is an Omnispeak-based version of Keen 7.

It shows two separate ways of making an Omnispeak mod:
- MOD.CK4 includes all of the changes to Omnispeak.
- EPISODE.CK4, ACTION.CK4, STRINGS.CK4 are pre-modified.

To run, either:
	Run path-to-omnispeak/omnispeak /episode mod.ck4
	from this directory.

	- OR -

	Copy these files into the main Omnispeak directory
	(overwriting if necessary) and then run omnispeak.

Feel free to report any bugs on the Omnispeak bugtracker:
	https://github.com/sulix/omnispeak/issues
