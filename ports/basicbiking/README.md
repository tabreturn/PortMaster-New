## Controls

| Button     | Action      |
| ---------- | ----------- |
| L-STICK    | Mouse move  |
| X          | Mouse click |
| LEFT/RIGHT | Pedals      |
| or L1/R1   | Pedals      |
| or B/A     | Pedals      |
| SELECT     | Restart     |
