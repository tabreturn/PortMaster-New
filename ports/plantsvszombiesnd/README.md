## Notes

Thanks to CrisDevelop for developing this game and kindly allowing us to host it on PortMaster.

**Note for devices with 1GB RAM:**

The game will slowly fill memory as you play, so it is recommended to use ZRAM if available or restart the game if you start to see performance issues.

## Controls

| Button | Action |
|--|--| 
|DPAD / Left Stick|Mouse movement|
|A|Left mouse button|
|B|Right mouse button|
|X|Mouse |
