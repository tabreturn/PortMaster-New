Compatibility and System Requirements:

No sound ingame on Windows? Install OpenAL, included with the package. (oalinst.exe)
Linux users should install package "libopenal1"

For Optimal performance, please make sure you have all your drivers up to date and correctly configured!

Minimum requirements:
CPU - 1.60GHz Atom or better equivalent.
RAM - 512MB minimum.
Graphics card - Anything with at least 64MB of Video RAM and has OpenGL support.
Storage - At least 5MB of free HDD space.

OS:

Windows Vista or newer - Windows XP is compatible but not supported.
Ubuntu 12.10 or newer - x64 users will be required to install the relevant 32-bit packages (package ia32libs) and the OpenAL library (package libopenal1)

Game Controls - Keyboard and Gamepad:

Arrow Keys or Joystick X/Y Axis(Left Stick) - Movement
SPACE or Controller Button 1(A Button) - Confirm / Action / Pass Turn
SHIFT or Controller Button 2(B Button) - Cancel / Enter Menu
F12 - Take Screenshot (they are saved in the same folder as the game executable)
ESC - Fast Quit (save and exit direct to OS, will ask to confirm)
