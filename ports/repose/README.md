## Controls

| Button | Action            |
| ------ | ----------------- |
| D-PAD  | Movement          |
| SELECT | Inventory         |
| A      | Equip handgun     |
| B      | Use               |
| X      | Equip axe         |
| Y      | Equip light stick |
