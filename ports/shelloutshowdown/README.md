## Controls

| Button | Action         |
| ------ | -------------- |
| D-PAD  | Move           |
| A      | Dash / cancel  |
| B      | Jump / confirm |
| Y      | Throw          |
| X      | Chat           |
| L      | Toggle name    |
| START  | Start / pause  |
