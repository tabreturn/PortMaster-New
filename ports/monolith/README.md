# Controls

| Button  | Action            |
| ------- | ----------------- |
| L-STICK | Move              |
| L3      | Fire              |
| R-STICK | Mouse / aim       |
| R1      | Dash              |
| R2      | Bomb              |
| START   | Pause             |
| A       | Confirm           |
