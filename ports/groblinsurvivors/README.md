## Controls

| Button        | Action       |
| ------------- | ------------ |
| D-PAD/L-STICK | Movement     |
| R-STICK       | Mouse cursor |
| A/L1          | Left click   |
| B/R1          | Right click  |
| Y             | Pause        |
| X             | Space key    |
