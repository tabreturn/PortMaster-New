## Notes
Wizball began as a 1987 Sensible Software game designed by Jon Hare and Chris Yates, with music by Martin Galway. In 2006-2007, Graham Goring and collaborators created the Retrospec remake for Windows and Mac, keeping the strange, colourful spirit of the original while giving it a more modern presentation.
 
## Controls

| Button | Action |
|--|--|
|Select| ESC|
|Start| 1 (Start 1 Player Game)|
|A| Select Powerup|
|B| Main Weapon|
|Dpad| Movement|
