## Controls

| Button | Action |
|--|--| 
|DPAD|Menu navigation|
|DPAD Left| Health Potion|
|Left Stick|Movement|
|A|Confirm / Jump|
|B|Back / Conjure|
|X|Sword Attack|
|Y|Interaction with certain objects|
|R1|Dash|
|R2|Long Jump|
|L1|Change Throw|
|L2|Map|
|Select|Inventory|
|Start|Pause menu|
